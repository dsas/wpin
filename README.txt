Displays [pinboard.in](https://pinboard.in) bookmarks using shortcode

== Description ==

WPinboard provides a shortcode to display bookmarks from [https://pinboard.in](https://pinboard.in)

The shortcode to use is [wpin] and it must enclose a pinboard.in url,
for example [wpin]https://pinboard.in/popular/[/wpin] or [wpin]https://pinboard.in/u:USERNAME/[/wpin]
